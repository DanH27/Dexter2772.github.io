Blog Post 2 - Sunday, October 8, 2017

This week in class we really started to get into responsive web design. I learned that this is very important as the number of people who use their phones to browse the web has overtaken the number who use their laptops/desktops. This is why it is important to ensure that the website looks just as good on a phone as it will on computer screen. We also started to learn about how you add media queries to your css file in order to apply a certain style set when the specifications of the query are met or not met. I got to experiment with the flexbox property for laying out my webpage and now have a better understanding of it. I am hoping to use this information that I learned in order to start improving all the projects that I make by using flexbox instead of non responsive CSS styles. And next week we are going to start learning bootstrap.

This week we continued to work on our project for the Lancaster Tech Pages. I mostly did media queries this week because some of them were not working properly and the content exceeded the container when it shifted. Other problems I was having when getting the “hamburger bar” to show up when it got to mobile view. I found a “hamburger bar” that uses pure HTML/CSS instead of using JavaScript. Now that I completed my projects I have the habit of checking every website I go to and see if it is responsive or not.
